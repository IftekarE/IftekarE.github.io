Quadratic Equation Program

This program can solve for

  • Axis of symmetry
  • vertex
  • Y-intercept
  • Type of roots
  • Number of roots
  • Roots
  • It can also graph the equation
Calculator Program
This is a basic calculator program. The Maximum value number you can enter is 99999999 and the minimum is -99999999 anything more or less than that and the calculator will say overloaded. The top left-hand display is for the first term the top right display is for the sign and the bottom left display is for the solution and the term currently being entered. Entering another number after the calculation has been done will reset and clears everything.
Overhead Shooter

The objective of this game is to shoot down 10 hostiles, every hostile you shoot down will reward you 50 points; You have 3 lives to complete the objective.

Controls

  • Click green flag to start
  • Standard "WASD" key movement
  • "Space" key to shoot
  • "R" key to reload
